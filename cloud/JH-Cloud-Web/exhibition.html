辽宁高分中心落户省气象局：2015年8月7日，在中国气象局与辽宁省人民政府签署省部合作协议的座谈会上，中国气象局郑国光局长提出在辽宁省气象局建设高分辨率对地观测系统数据与应用中心的意向，辽宁省陈求发省长也提出了在气象部门加强国家高分辨率对地观测系数据应用的建议，最终郑国光局长和陈求发省长就由辽宁省气象局申请成立“高分辨率对地观测系统辽宁省数据与应用中心”（以下简称高分中心）达成一致。 辽宁省气象局代辽宁省政府草拟了《关于申请成立高分辨率对地观测系统辽宁省数据与应用中心的函》（辽政函〔2015〕82号），2015年11月16日上报国防科技工业局，2016年1月10日国家国防科技工业局批复，同意依托辽宁省气象局设立高分中心。赵化明副省长在该文件上批示“气象局抓好基础建设，人员培训，工作对接，尽快服务于全省经济社会建设。
